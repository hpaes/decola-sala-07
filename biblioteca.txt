01. git rm - deleta arquivo
02. git checkout - alternância de branch
03. git commit -am "nome do commit" - adiciona e commita o arquivo
04. git branch "nome da branch" - cria uma nova branch
05. git status - verifica o estado do repositório
06. git push origin master
07. git cd +diretório - muda a pasta atual para a pasta do diretório
	  /diretório  - da pasta atual para a pasta do diretório
	  .. 	      - volta uma pasta
